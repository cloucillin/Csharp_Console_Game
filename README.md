# Csharp_Console_Game C#控制台实现主角移动打boss营救公主的小游戏
#面向流程的 游戏编程：逻辑性强

#主要代码内容位于program.cs中

#玩家进入游戏输入自己的名字按下回车进入开始游戏选项

#ws键调整当前选择项

#进入游戏后刷新主角和boss位置位于规定的位置在红墙内对话框之上

#用户可以通过wasd来操控主角在舞台的合法范围内移动

#boss在两回合后会随机朝一个随机方向移动一次

#在移动过程中任何角色不会超出红墙且不会重叠在一起

#主角在靠近boss后可以按下j键与之战斗，期间双方不会移动，主角和boss的攻击力为区间内随机值

#不断按下J键打败boss或者战死

#战死会来到结束界面选择再次进入游戏或者退出

#打败boss会刷新公主靠近公主按下j键即可营救公主完成游戏 

#来到结束界面可选择再次进入游戏


#通过改变光标的坐标来实现绘制各个游戏元素

#开始界面-控制台输出-控制台输入-控制台颜色变化

#游戏界面-控制台输出-控制台输入-控制台颜色变化-while-switch

#判断 - 条件运算符 - if语句

#回合制战斗 - 随机数 - 循环 - if语句

#结束界面 - 控制台输出 - 控制台输入 - 控制台颜色变化

#界面之间的相关切换用nowSceneId来控制不同场景的切换
